# Происхождение набора данных

Набор `data/raw/educational_services_prices.csv` формируется скриптом
`python data/build_dataset.py`. Случайная генерация значений не используется.
Каждая строка исходного CSV содержит `source_url`, номер исходной строки и
название публикации.

## Открытые источники

1. **Роспатент — платные образовательные услуги ФГБОУ ВО РГАИС.**
   Официальный набор содержит названия программ, уровень образования, форму
   обучения и стоимость. Скрипт использует опубликованные версии от
   19.10.2017, 29.06.2021, 05.09.2022, 04.10.2023 и 02.09.2024:
   <https://rospatent.gov.ru/opendata/7730176088-paidservices>.
2. **РУДН — численность студентов по направлениям подготовки.**
   Официальный открытый CSV используется как внешний показатель спроса:
   <https://opendata.rudn.ru/od/programs>.
3. **Всемирный банк — Consumer price index for the Russian Federation.**
   Открытый API `FP.CPI.TOTL` используется как проверяемый макроэкономический
   индекс, перебазированный к 2017 году:
   <https://api.worldbank.org/v2/country/RUS/indicator/FP.CPI.TOTL?format=json>.

## Формирование признаков

- `final_price` — опубликованная Роспатентом стоимость услуги.
- `base_price` — предыдущая опубликованная цена той же услуги и формы обучения;
  для первой записи используется медиана сопоставимой группы.
- `competitor_price` — медиана цен того же года, уровня и формы обучения.
- `student_count` — открытая численность студентов РУДН по совпадающему коду
  направления; для отсутствующих кодов применяется медиана опубликованных
  значений.
- `demand_index` — нормированная в диапазон 20–100 численность студентов.
- `salary_index` — индекс потребительских цен Всемирного банка, используемый
  как макроэкономический прокси и перебазированный к 2017 году.
- `duration_months` — нормативная продолжительность по уровню образования.
- `advertising_spend` и `discount_percent` равны нулю, поскольку эти сведения
  отсутствуют в открытых источниках. Это явно заданное отсутствие данных, а не
  случайно сгенерированное значение.
- `region` равен `Москва`, поскольку РГАИС находится в Москве.

Обработанный файл `data/processed/prepared_dataset.csv` содержит только признаки,
которые использует ML-модуль. Исходный файл дополнительно хранит поля
происхождения данных для аудита и воспроизводимости.
