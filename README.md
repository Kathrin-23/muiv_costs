# Информационная система «Анализ и прогнозирование ценообразования на образовательные услуги»

Проект представляет собой веб-приложение на Python и Flask для загрузки данных об образовательных услугах, анализа динамики цен, обучения модели машинного обучения и прогнозирования рекомендуемой стоимости образовательной услуги.

## Состав проекта

- `main.py` — файл запуска приложения.
- `app/routes.py` — все маршруты веб-приложения.
- `app/models.py` — все модели базы данных.
- `app/ml_service.py` — сервис подготовки данных, обучения модели и прогнозирования.
- `app/report_service.py` — формирование отчетов в форматах XLSX и DOCX.
- `database/education_pricing.db` — демонстрационная база данных.
- `data/raw/educational_services_prices.csv` — исходный набор данных.
- `ml/price_model.pkl` — обученная модель прогнозирования цены.

## Быстрый запуск

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

На macOS/Linux обычно используется команда `python3`:

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
```

После запуска приложение будет доступно по адресу: `http://127.0.0.1:5000`.

## Основные функции

1. Авторизация и регистрация пользователей.
2. Личный кабинет пользователя.
3. Каталог образовательных услуг.
4. Анализ истории цен.
5. Загрузка набора данных в формате CSV.
6. Обучение модели прогнозирования.
7. Прогнозирование стоимости образовательной услуги.
8. Формирование отчетов.
9. Форма обратной связи.
10. Администрирование пользователей, ролей, программ, услуг, датасетов, отчетов, обращений и настроек.
11. Личный кабинет с прогнозами, отчетами, загруженными данными и настройками профиля.
12. Справка по системе.
